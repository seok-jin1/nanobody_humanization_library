# IMGT Numbering Ranges for Nanobody (V-DOMAIN)

IMGT 표준 넘버링 시스템에 따른 항체 가변 영역(V-DOMAIN)의 Framework(FR) 및 Complementarity Determining Regions(CDR) 범위는 다음과 같습니다.

| 영역 (Region) | IMGT 범위 (Positions) | 설명 |
| :--- | :--- | :--- |
| **FR1-IMGT** | 1 - 26 | Framework region 1 |
| **CDR1-IMGT** | 27 - 38 | Complementarity determining region 1 |
| **FR2-IMGT** | 39 - 55 | Framework region 2 |
| **CDR2-IMGT** | 56 - 65 | Complementarity determining region 2 |
| **FR3-IMGT** | 66 - 104 | Framework region 3 |
| **CDR3-IMGT** | 105 - 117 | Complementarity determining region 3 |
| **FR4-IMGT** | 118 - 128 | Framework region 4 (J-REGION 일부 포함) |

## 참고 사항
- **Insertion Codes:** CDR3 영역이 표준 길이(13개 잔기)보다 길 경우, 111번과 112번 사이에 알파벳(111.1, 111.2... 또는 PDB 형식의 111A, 111B...)이 추가될 수 있습니다.
- **Nanobody (VHH) 특이성:** Nanobody는 일반적인 Heavy chain 가변 영역(VH)의 IMGT 넘버링 체계를 그대로 따릅니다.
- **V-DOMAIN 종료:** 일반적으로 V-DOMAIN은 128번에서 종료되는 것으로 간주됩니다.
